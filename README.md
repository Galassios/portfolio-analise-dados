# Análise de Salário Médio por Área

## Objetivo
Analisar o salário médio e a quantidade de pessoas por área a partir de uma base de dados fictícia.

## Etapas da análise
- Leitura do arquivo CSV
- Remoção de registros duplicados
- Agrupamento por área
- Cálculo da média salarial e contagem de pessoas
- Ordenação dos resultados

## Resultado
Foi possível identificar quais áreas apresentam maior salário médio e maior concentração de colaboradores.

## Tecnologias
Python, Pandas, SQL
