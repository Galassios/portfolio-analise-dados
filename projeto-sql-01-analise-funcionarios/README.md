# Projeto SQL 01 — Análise de Funcionários

Este projeto simula análises realizadas por um analista de dados em uma empresa fictícia.

O objetivo é explorar dados de funcionários para gerar insights sobre salários, áreas da empresa e contratações ao longo do tempo.

## Dataset

O dataset contém informações sobre funcionários da empresa:

- id
- nome
- area
- salario
- data_contratacao

## Perguntas respondidas

As análises incluem:

- Média salarial por área
- Ranking salarial dentro de cada área
- Top 2 salários por área
- Funcionários acima da média da área
- Percentual de funcionários por área
- Contratações por ano
- Acumulado de contratações

## Técnicas SQL utilizadas

- GROUP BY
- HAVING
- CASE
- Subqueries
- CTE
- Window Functions
- RANK()
- SUM() OVER()
- LAG()
